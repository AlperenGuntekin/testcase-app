
# TestCase App - Next.js

Some projects I've developed using Next.js




## Run it on your computer

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install required packages

```bash
  npm install
```

Run the server

```bash
  npm run start
```
